# بوت Discord لمراقبة قنوات TikTok

بوت Discord متقدم يقوم بمراقبة قنوات TikTok وإرسال إشعارات تلقائية عند نشر فيديوهات جديدة أو بدء بث مباشر.

## المميزات

- مراقبة قنوات TikTok المتعددة
- إشعارات فورية عند نشر فيديو جديد
- إشعارات عند بدء البث المباشر
- حماية بكلمة سر
- تخزين البيانات في قاعدة بيانات Supabase
- أوامر سلاش باللغة العربية

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
npm install
```

2. تشغيل البوت:
```bash
npm start
```

## إضافة البوت إلى السيرفر

لإضافة البوت إلى سيرفر Discord الخاص بك، اتبع الخطوات التالية:

### الخطوة 1: الحصول على رابط الدعوة

1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. اختر تطبيق البوت الخاص بك (أو قم بإنشاء واحد جديد)
3. من القائمة الجانبية، اختر **OAuth2** → **URL Generator**

### الخطوة 2: اختيار الصلاحيات

في صفحة URL Generator، اختر الصلاحيات التالية:

**في قسم SCOPES:**
- ✅ `bot`
- ✅ `applications.commands`

**في قسم BOT PERMISSIONS:**
- ✅ Send Messages (إرسال الرسائل)
- ✅ Mention @everyone, @here, and All Roles (منشن للجميع)
- ✅ Read Messages/View Channels (قراءة الرسائل)
- ✅ Embed Links (إرسال روابط مدمجة)
- ✅ Use Slash Commands (استخدام أوامر السلاش)

### الخطوة 3: نسخ الرابط وإضافة البوت

1. انسخ الرابط الذي تم إنشاؤه في أسفل الصفحة
2. الصق الرابط في المتصفح
3. اختر السيرفر الذي تريد إضافة البوت إليه
4. اضغط على **Authorize** (تفويض)
5. أكمل عملية التحقق من الأمان (CAPTCHA)

### رابط الدعوة السريع

إذا كنت تريد رابط دعوة جاهز، استخدم هذا الرابط (استبدل `YOUR_CLIENT_ID` بمعرف العميل الخاص بك):

```
https://discord.com/api/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=2147502080&scope=bot%20applications.commands
```

للحصول على معرف العميل (Client ID):
1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. اختر تطبيق البوت
3. في صفحة **General Information**، انسخ **Application ID**

## الأوامر المتاحة

### `/اضافة_قناة`
إضافة قناة TikTok جديدة للمراقبة

**خطوات الاستخدام:**
1. استخدم الأمر `/اضافة_قناة`
2. أدخل كلمة السر: `#OneSouLeSports`
3. أدخل يوزرنيم قناة TikTok (مثال: `mo2026_editor`)
4. أدخل رسالة الفيديو الجديد (مثال: "نزل فيديو جديد 🔥")
5. أدخل رسالة البث المباشر (مثال: "القناة بدأت لايف الآن 🔴")
6. اختر قناة Discord التي سيتم النشر فيها

### `/القنوات`
عرض جميع قنوات TikTok التي يتم مراقبتها

### `/حذف [يوزرنيم]`
حذف قناة TikTok من المراقبة

**مثال:**
```
/حذف mo2026_editor
```

## الأمان

- جميع الأوامر المهمة محمية بكلمة سر
- كلمة السر الافتراضية: `#OneSouLeSports`
- يتم حذف رسالة كلمة السر فوراً بعد إدخالها

## آلية العمل

- يقوم البوت بفحص قنوات TikTok كل دقيقتين
- عند اكتشاف فيديو جديد أو بث مباشر، يرسل إشعار فوري
- يتم تخزين جميع البيانات في قاعدة بيانات Supabase

## المتطلبات التقنية

- Node.js 16 أو أحدث
- حساب Discord للمطورين
- حساب Supabase (مجاني)
- توكن بوت Discord صالح

## الدعم

في حالة وجود مشاكل، تأكد من:
- البوت لديه جميع الصلاحيات المطلوبة
- التوكن صحيح في ملف `.env`
- الاتصال بالإنترنت مستقر
- قاعدة البيانات تعمل بشكل صحيح
